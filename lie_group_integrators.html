Lie group integrators¶

The formulation of differential equations on manifolds¶

Lie group integrators solve differential equations whose solution evolve on a manifold \(\mathcal{M}\). For ease of notation we restrict the discussion to the case of autonomous vector fields, although allowing for explicit \(t\)-dependence could easily have been included. This means that we seek a curve \(y(t)\in\mathcal{M}\) whose tangent at any point coincides with a vector field \(F\in\mathcal{X}(\mathcal{M})\) and passing through a designated initial value \(y_0\) at \(t=t_0\)

(1)¶\[\begin{align} \dot{y}(t) = F|_{y(t)},\qquad y(t_0)=y_0. \end{align}\]

Before addressing numerical methods for solving <eq:11> it is necessary to introduce a convenient way of representing the vector field \(F\). There are different ways of doing this. %One is to furnish \(\mathcal{M}\) with a transitive action by some Lie group \(G\) of dimension \(d\geq\dim \mathcal{M}\). One is to furnish \(\mathcal{M}\) with a transitive action \(\Psi: G \times \mathcal{M} \rightarrow \mathcal{M}\) by some Lie group \(G\) of dimension \(d\geq\dim \mathcal{M}\). We denote the action of \(g\) on \(m\) as \(g\cdot m\), i.e. \(g\cdot m=\Psi(g,m)\). Let \(\g\) be the Lie algebra of \(G\), and denote by \(\exp: \g\rightarrow G\) the exponential map. We define \(\Psi_*:\g\rightarrow\mathcal{X}(\mathcal{M})\) to be the infinitesimal generator of the action, i.e.

(2)¶\[\begin{align} \left.F_\xi\right|_m= \left.\Psi_*(\xi)\right|_m = \left.\frac{d}{dt}\right|_{t=0} \Psi(\exp(t\xi), m) \end{align}\]

The transitivity of the action now ensures that \(\left.\Psi_*(\g)\right|_m=T_m\mathcal{M}\) for any \(m\in\mathcal{M}\), such that any tangent vector \(v_m\in T_m\mathcal{M}\) can be represented as \(v_m=\left.\Psi_*(\xi_v)\right|_m\) for some \(\xi_v\in\g\) (\(\xi_v\) may not be unique). Consequently, for any vector field \(F\in\mathcal{X}(\mathcal{M})\) there exists a map \(f:\mathcal{M}\rightarrow\g\)footnote{RE{If the Lie group action is smooth, a map \(f\) of the same regularity as \(F\) can be found cite{warner83fod} }} such that

(3)¶\[\begin{align} F|_m = \left.\Psi_*(f(m))\right|_m,\quad\text{for all}\; m\in \mathcal{M} \end{align}\]

This is the original tool cite{munthe-kaas99hor} for representing a vector field on a manifold with a group action. Another approach was used in cite{crouch93nio} where a set of {em frame vector fields} \(E_1,\ldots, E_d\) in \(\mathcal{X}(\mathcal{M})\) was introduced assuming that for every \(m\in \mathcal{M}\),

(4)¶\[\begin{align} \text{span}\{\left.E_1\right|_m,\ldots,\left.E_d\right|_m\}= T_m \mathcal{M}. \end{align}\]

Then, for any vector field \(F\in\mathcal{X}(\mathcal{M})\) there are, in general non-unique, functions \(f_i:\mathcal{M}\rightarrow \mathbb{R}\), RE{which can be chosen with the same regularity as \(F\),} such that

(5)¶\[\begin{align} F|_m = \sum_{i=1}^d f_i(m) \left.E_i\right|_m. \end{align}\]

A fixed vector \(\xi\in\mathbb{R}^d\) will define a vector field \(F_\xi\) on \(\mathcal{M}\) similar to eqref{frozenaction}

(6)¶\[\begin{align} \left.F_{\xi}\right|_m = \sum_{i=1}^d \xi_i E_i|_m \end{align}\]

If \(\xi_i=f_i(p)\) for some \(p\in\M\), the corresponding \(F_\xi\) will be a vector field in the linear span of the frame which coincides with \(F\) at the point \(p\). Such a vector field was named by cite{crouch93nio} as a emph{the vector field frozen at \(p\)}.

The two formulations just presented are in many cases connected, and can then be used in an equivalent manner. Suppose that \(e_1,\ldots,e_d\) is a basis of the Lie algebra \(\g\), then we can simply define frame vector fields as \(E_i = \Psi_*(e_i)\) and the vector field we aim to describe is,

(7)¶\[\begin{align} F|_m=\left.\Psi_*(f(m))\right|_m= \left.\Psi_*(\sum_i f_i(m)e_i)\right|_m=\sum_i f_i \left.E_i\right|_m. \end{align}\]

As mentioned above there is a non-uniqueness issue when defining a vector field by means of a group action or a frame. A more fundamental description can be obtained using the machinery of connections. The assumption is that the simply connected manifold \(\mathcal{M}\) is equipped with a connection which is flat and has constant torsion. Then \(F_p\), the frozen vector field of \(F\) at \(p\) defined above, can be defined as the unique element \(F_p\in\mathcal{X}(\mathcal{M})\) satisfying begin{enumerate} item \(F_p|_p=F|_p\) item \(\nabla_X F_p=0\) for any \(X\in\mathcal{X}(\mathca{M})\). end{enumerate} So \(F_p\) is the vector field that coincides with \(F\) at \(p\) and is parallel transported to any other point on \(\mathcal{M}\) by the connection \(\nabla\). Since the connection is flat, the parallel transport from the point \(p\) to another point \(m\in\mathcal{M}\) does not depend on the chosen path between the two points. For further details, see e.g. cite{lundervold15oas}.
